# ⚡ Smart Energy Meter using ESP32 and Blynk

## 📌 Overview
The Smart Energy Meter is an IoT-based project that monitors electrical parameters such as voltage, current, power, and energy consumption in real time using an ESP32 microcontroller. The measured data is displayed on an LCD and sent to the Blynk IoT platform for remote monitoring.

---

## ✨ Features
- Real-time voltage monitoring
- Real-time current monitoring
- Power calculation
- Energy consumption monitoring
- Cost estimation
- LCD display
- Blynk IoT remote monitoring
- Overload detection
- Telegram alert notification
- ESP32 Wi-Fi connectivity

---

## 🛠 Hardware Components
- ESP32 Development Board
- ACS712 Current Sensor
- ZMPT101B Voltage Sensor
- 16x2 I2C LCD Display
- Relay Module
- Buzzer
- Breadboard
- Connecting Wires
- Power Supply

---

## 💻 Software Used
- Arduino IDE
- Blynk IoT
- Embedded C/C++

---

## ⚙️ Working Principle
1. ESP32 reads voltage and current from the sensors.
2. The controller calculates power and energy consumption.
3. Readings are displayed on the LCD.
4. Data is sent to the Blynk application through Wi-Fi.
5. If abnormal conditions are detected, the relay disconnects the load and a Telegram alert is sent.
